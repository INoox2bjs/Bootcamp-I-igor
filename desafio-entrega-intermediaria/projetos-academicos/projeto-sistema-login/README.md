# 🔐 Sistema de Login

Projeto de sistema de login desenvolvido utilizando HTML, PHP, CSS, JavaScript e banco de dados.

Criado com o objetivo de praticar conceitos de desenvolvimento web full stack, como:

- Validação de formulários
- Autenticação de usuários
- Integração com banco de dados
- Manipulação de sessões
- Organização de interface web

## 🚀 Linguagens

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## ▶️ Funcionalidades

- Cadastro de usuários
- Login e autenticação
- Validação de campos
- Integração com banco de dados

## 📄 Objetivo

Praticar desenvolvimento full stack e aprimorar conhecimentos em sistemas web.
